제1과목: 전기이론 (핵심 총정리)

모바일 최적화 / 초정밀 요약본

1. 직류회로 (DC Circuit)

기초 법칙

  • 전하량(Q): Q = I · t = C · V [C]
  • 전류(I): 단위시간당 흐르는 전하량 I = Q / t [A]
  • 전압(V): 전위차, V = W / Q (W: 일, Joule)
  • 옴의 법칙: V = I · R, I = V / R, R = V / I
  • 저항(R): R = ρ · (L / A) (ρ: 고유저항, L: 길이, A: 단면적)
  • 컨덕턴스(G): 저항의 역수 G = 1 / R [℧, S]

저항의 접속

  • 직렬접속: Rt = R1 + R2 (전류 일정, 전압 분배)
  • 전압분배: V1 = {R1 / (R1 + R2)} · V
  • 병렬접속: Rt = (R1 · R2) / (R1 + R2) (전압 일정, 전류 분배)
  • 전류분배: I1 = {R2 / (R1 + R2)} · I (상대방 저항 비례)

전력과 열량

  • 전력(P): P = VI = I²R = V²/R [W]
  • 전력량(W): W = P · t = Pt [J, Wh]
  • 줄의 법칙(H): H = 0.24 · I²Rt [cal]
  • 마력(HP): 1 HP = 746 [W]

2. 정전기 (Electrostatics)

전하와 전계

  • 쿨롱의 법칙: F = 9x10⁹ · (Q1Q2 / r²) [N]
  • 전계의 세기(E): E = F / Q = 9x10⁹ · (Q / r²) = V / d [V/m]
  • 전위(V): V = 9x10⁹ · (Q / r) [V]
  • 가우스의 정리: 전속수 N = Q / ε

콘덴서 (Capacitor)

  • 정전용량(C): C = ε · (A / d) (ε: 유전율)
  • 유전율(ε): ε = ε0 · εr (ε0: 진공유전율 8.854x10⁻¹²)
  • 콘덴서 접속: 직렬(합분의 곱), 병렬(단순 합) ※저항과 반대
  • 정전 에너지: W = 1/2 CV² = 1/2 QV = Q²/2C [J]
  • 공기 유전율: 비유전율 εr = 1

3. 자기현상 (Magnetism)

자계와 자기력

  • 자기의 쿨롱 법칙: F = 6.33x10⁴ · (m1m2 / r²) [N]
  • 자계의 세기(H): H = F / m [AT/m]
  • 자속밀도(B): B = μH = Φ / A (μ: 투자율)
  • 기자력(F): F = NI = RΦ [AT]
  • 자기저항(R): R = L / (μA)

전자력과 유도기전력

  • 플레밍 왼손: 전동기 (검지:자기장, 중지:전류, 엄지:힘)
  • 전자력의 크기: F = BIl sinθ
  • 플레밍 오른손: 발전기 (검지:자기장, 엄지:속도, 중지:기전력)
  • 유도기전력: e = Blv sinθ
  • 패러데이 법칙: e = -N · (ΔΦ / Δt) (렌츠의 법칙: 방향 -)

인덕턴스

  • 자기 인덕턴스(L): L = NΦ / I [H]
  • 축적 에너지: W = 1/2 LI² [J]
  • 결합 계수(k): k = M / √(L1L2) (0 ≤ k ≤ 1)
  • 인덕턴스 접속: 가동합성 L1+L2+2M, 차동합성 L1+L2-2M

4. 교류 회로 (AC Circuit)

교류의 표시

  • 순시값: i = Im sin(ωt + θ)
  • 각속도(ω): ω = 2πf (60Hz일 때 약 377)
  • 실효값(RMS): V = Vm / √2 (최댓값의 0.707배)
  • 평균값(Avg): Va = 2Vm / π (최댓값의 0.637배)
  • 파고율: 최/실 (최댓값/실효값), 파형율: 실/평 (실효값/평균값)

R-L-C 회로

  • R회로: 전압-전류 동위상
  • L회로: 전류가 90° 느림(지상), XL = 2πfL
  • C회로: 전류가 90° 빠름(진상), XC = 1 / (2πfC)
  • 임피던스(Z): Z = √(R² + (XL-XC)²)
  • 공진 주파수: f = 1 / (2π√(LC))

교류 전력

  • 피상전력(Pa): Pa = VI = I²Z [VA]
  • 유효전력(P): P = VI cosθ = I²R [W]
  • 무효전력(Pr): Pr = VI sinθ = I²X [Var]
  • 역률(cosθ): cosθ = P / Pa = R / Z

5. 다상 교류 (3-Phase AC)

결선과 전력

  • Y결선(성형): Vl = √3Vp, Il = Ip (중성점 접지 가능)
  • Δ결선(환형): Vl = Vp, Il = √3Ip (제3고조파 순환)
  • 3상 전력: P = √3 Vl Il cosθ = 3 Vp Ip cosθ
  • 2전력계법: P = W1 + W2, Pa = 2√(W1²+W2²-W1W2)

비정현파 및 과도현상

  • 비정현파 구성: 직류분 + 기본파 + 고조파
  • 비정현파 실효값: V = √(V0² + V1² + V2² + ...)
  • 시상수(τ): RL직렬 τ = L/R, RC직렬 τ = RC
